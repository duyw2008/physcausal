# 压缩技术分类学

> 位置: information/ | PhysCausal Agent
> 核心命题: 压缩即理解。智能体 = 在熵增宇宙中做局域熵减的机器。

---

## 统一原理

所有压缩技术是同一种数学操作在不同空间中的投影：

```
从可能性空间 S 中，保留满足约束 C 的最小子空间 S|C

路径         空间 S              约束 C                  子空间 S|C
──────────────────────────────────────────────────────────────────
谱分解       数据矩阵             最大方差                 主成分
信息论       概率分布             最小码率                 最优编码
变分         后验分布簇           KL 散度 min              最紧后验
稀疏         模型空间             非零元最少               L1 稀疏解
学习         权重空间             重建误差 min             瓶颈表示
量子         Hilbert 空间         测量基                   本征态
物理         所有可能路径         作用量 min               真实路径
```

## 七条技术路径

### 1. 谱分解类 — 找「最重要的方向」

```
本质: 对矩阵做特征分解，保留最大特征值对应的方向

PCA         协方差矩阵 → 最大方差方向
            实现: spectral/spectral.py ✅

SVD         任意矩阵 → 最佳低秩近似 (Eckart-Young 定理)
            X ≈ U_k Σ_k V_k^T 是所有秩-k 近似中 Frobenius 误差最小的
            实现: spectral/spectral.py ✅

Koopman     非线性动力学 → 线性化谱
            K φ_k = μ_k φ_k
            实现: spectral/spectral.py ✅

拉普拉斯谱   图 L=D-A 的特征分解 → 谱聚类
            λ₂ = 代数连通度, 谱间隙 = 最优聚类数
            实现: spectral/spectral.py ✅

ICA         最大化统计独立性 (非高斯信号分离)
            X = AS, 找 S 的各分量独立

NMF         非负矩阵分解 → 部件级别的表示
            X ≈ WH, W≥0, H≥0

随机投影    Johnson-Lindenstrauss 引理
            高维点 → 随机投影 → 保距低维
            O(n log n) 极快
```

### 2. 信息论类 — 度量「还剩多少信息」

```
本质: 用概率分布的熵来量化信息量

Shannon 熵   H(X) = -Σ p(x) log p(x)
             不确定性的度量
             实现: information/shannon.py ✅

互信息       I(X;Y) = H(X) + H(Y) - H(X,Y)
             X 和 Y 共享的信息
             实现: information/shannon.py ✅

传递熵       TE(X→Y) = I(Y_t ; X_{t-1} | Y_{t-1})
             Granger 因果的信息论版本
             实现: information/shannon.py ✅

KL 散度      D_KL(P||Q) = Σ p log(p/q)
             分布差异的度量
             实现: information/shannon.py ✅

信息瓶颈     min I(T;X) s.t. I(T;Y) ≥ c
             感知压缩的理论基础
             实现: information/bottleneck.py ✅

率失真       min I(X;Z) s.t. E[d(X,Z)] ≤ D
             给定失真下界的最小信息率

MDL          最小描述长度
             Kolmogorov 复杂度的实用近似
```

### 3. 变分/贝叶斯类 — 更新「信念」

```
本质: 用新数据更新先验分布，后验比先验更「紧」

贝叶斯推断   P(h|D) = P(D|h) P(h) / P(D)
             先验 × 似然 → 后验 (概率空间收缩)

变分推断     min KL(q(h)||p(h|D))
             在简单分布族中找最近似后验的

最大熵       max H(p) s.t. E[f_i(x)] = c_i
             Jaynes: 最不偏颇的分布
             实现: information/jaynes.py ✅

EM 算法      E步: Q(h) = P(h|D,θ_old)
             M步: θ_new = argmax E[log P(D,h|θ)]
             隐变量模型的最优压缩

自由能       F = ⟨E⟩ - TS
             能量与熵的竞争 → 平衡态 = 最优压缩
```

### 4. 稀疏/因果类 — 删除「不需要的边」

```
本质: 用更少的变量和更少的边来解释数据

L1 正则化    min loss + λ‖w‖₁
             产生稀疏解 (绝大多数权重 = 0)

因果发现     PC/FCI/GES: 稠密关联图 → 稀疏 DAG
             条件独立性检验 + 评分搜索
             实现: causal/discovery.py ✅

符号回归     数据 → 最短公式
             搜索表达式空间，Penalize 复杂度

DAG 拓扑序   因果图 → 偏序编码
             用最少的约束关系表示因果结构

张量分解     CP: X ≈ Σ λ_r · a_r ⊗ b_r ⊗ c_r
             Tucker: X ≈ G ×₁ A ×₂ B ×₃ C
             高维 → 低秩 — 高阶 PCA
```

### 5. 神经/学习类 — 训练「瓶颈」

```
本质: 迫使信息通过窄通道，只保留能重建的部分

自编码器     编码→瓶颈→解码
             L = ‖X - D(E(X))‖²

VAE          编码→μ,σ→采样→解码
             L = 重建误差 + D_KL(q(z|x)||p(z))
             变分瓶颈

VQ-VAE       编码→最近码本向量→解码
             将连续表示离散化为有限种可能

对比学习     maximize I(f(x); f(x')) / minimize I(f(x); f(y))
             SimCLR, MoCo: 丢掉不变量以外的信息

知识蒸馏     teacher(x) → student(x)
             大模型 → 小模型，保持预测能力

剪枝/量化    权重矩阵 → 稀疏(剪枝) 或 int8(量化)
             用更少的参数表示相同的函数
```

### 6. 量子类 — 观测即压缩

```
本质: 从叠加态中选出一个本征态

波函数坍缩   |ψ⟩ = Σ c_i|φ_i⟩ → |φ_k⟩
             测量从无限可能中选出一个确定结果
             实现: meta_physics/measurement.py ✅

退相干       环境与系统纠缠 → 叠加态消失
             不需要「观测者」— 环境就是观测者

DMRG         密度矩阵重整化群
             量子多体态的最优低秩表示

张量网络     MPS (矩阵积态), PEPS (投影纠缠对态)
             用量子纠缠结构编码经典信息
```

### 7. 物理类 — 自然界的压缩

```
本质: 物理定律本身就是宇宙的「压缩程序」

最小作用量   δS = 0
             从所有可能路径中选出真实路径
             实现: meta_physics/least_action.py ✅

重整化群     高能微观自由度 → 低能宏观有效理论
             不同尺度使用不同复杂度的模型
             这是「丢弃无关细节」的最纯粹形式

全息原理     黑洞熵 ∝ 面积 (不是体积!)
             三维世界的全部信息可以编码在二维表面

涌现         微观: 10²³ 个分子
             宏观: 温度/压力/密度 (3 个序参量!)
             宇宙史上最震撼的压缩比

对称性       Lagrangian 的对称性 → 守恒量
             一个不变性条件替换无数可能的方程形式
             实现: meta_physics/symmetry.py ✅
```

## 压缩的三个维度

不是所有压缩都一样。区分三种：

```
维度          问题                      例子
───────────────────────────────────────────────
丢弃噪声      丢掉什么不重要？          PCA 丢掉低方差成分
              重要性由本征值定义

保留结构      什么必须保留？            因果 DAG 保留因果边
              约束由物理/因果原则定义

重写编码      用更短的程序表达           SCM 的结构方程
              规律 = 最短的生成程序
```

对应 PhysCausal 的三层基础设施：

```
丢弃噪声 → spectral/      本征值 = 重要性
保留结构 → meta_physics/  五原则 = 约束
重写编码 → information/   描述长度 = 理解深度
```

# 物理直觉: 因果结构的共鸣

> PhysCausal 知识库 — 2026-06-09

## 什么是物理直觉

在因果链铺通之前，先感知到结构的共鸣。

物理直觉不是玄学，是对结构模式的快速匹配。就像棋手看一眼棋盘就能感知"这里危险"，不是因为数完了所有变化，而是模式库告诉他"这个结构和某个已知败局相似"。物理直觉做同样的事——在变量还没连上因果边之前，先认出"这个结构和另一个结构长得像"。

## PhysCausal 中的编码

类比引擎 (`creative/causal_analogy.py`) 把直觉编码为三个层次:

```
层次 1: 模块拓扑同构 — F=ma ↔ V=IR
         精确匹配, 绝对可靠, 极其稀少

层次 2: 因果链软匹配 — wavelength ↔ gauge_field (80%)
         结构剖面匹配, 不需要相同变量名
         这是当前的主阵地

层次 3: 种子联想 — entropy → wave_function ★
         手工编码的物理直觉, 不经过因果验证
         最不可靠, 但最大胆
```

三个层次对应不同的可靠性要求。层次 3 可以犯错, 层次 1 不能犯错, 层次 2 居中。

## 发现的结构共鸣

### 退相干 ≈ 经典耗散 (100%)

```
退相干链: environment_coupling → mixed_state → entropy → order_parameter
耗散链:   velocity → dynamic_pressure → drag_force → kinetic_energy_loss → entropy

共享结构:
  - 子系统 ↔ 环境边界
  - 序参量变化 → 熵增
  - 不可逆性来自子系统视角
```

物理含义: 量子退相干是经典摩擦在量子域的对应物。两者都是信息/能量从子系统逃逸到环境。

### 波长 ≈ 规范场 (80%)

```
波长链 (量子/声学): wavelength → wave_speed → interference_pattern
规范场链 (电磁/统一): gauge_field → lorentz_force → particle_creation

共享结构:
  - 波现象在不同域的投影
  - 从"传播"到"作用"的因果转换
  - 长度匹配 (2-3步)
```

物理含义: 如果粒子是波 (de Broglie), 力也是波 (规范场)。量子场论是这个直觉的严格形式化。

### 动量 ≈ 额外维度 (80%)

```
动量链 (声学/量子): momentum → kinetic_energy → temperature → ...
额外维链 (电磁/统一): compact_dimension → ...
```

物理含义: 额外维度的动量在 4 维表现为电荷 — Kaluza-Klein 理论。类比引擎在因果图刚铺了两步时就捕捉到了这个共鸣。

### 规范场 ≈ 时空曲率 (100%)

```
gauge_field → lorentz_force → particle_creation
spacetime_curvature → geodesic_path → tidal_force

共享结构:
  - 都是"力的几何化"
  - 都是从几何/场到粒子动力学的因果链
  - 长度和深度精确匹配
```

物理含义: Wheeler 的几何同一性 — 所有力都是几何。因果图在结构层面验证了这个直觉。

## Wheeler 的裂缝

> "前沿不在更小的尺度, 而在几何与物理的裂缝处。"

结构共鸣总是在这些裂缝处最强。因为:

- **裂缝** = 因果图还没铺通的区域
- **共鸣** = 两侧的结构已经在独立演化中对齐
- **直觉** = 感知到这种对齐, 但还没法写出精确的因果边
- **证明** = 把因果边铺通, 把直觉变成推导

这就是 δS=0 的工作方式——先在结构上对齐, 再在因果上验证。

## 使用指南

在 PhysCausal 的日常工作中:

```
1. analogy  → 发现结构共鸣 (自动化)
2. speculate → 基于共鸣生成大胆假说
3. focus    → 选定共鸣最强的方向深入
4. suggest --run → 交叉验证
5. paper    → 归档成果
```

共鸣是起点, 不是终点。但它是唯一不需要因果图完备就能工作的东西——这也是为什么物理直觉能在理论成熟之前几十年就"感觉"到正确方向。

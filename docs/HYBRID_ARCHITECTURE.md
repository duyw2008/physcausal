## 第一阶段: 因果链嵌入 (最小可行)

不需要神经网络。从因果图本身学习结构表示:

```
变量 → 图邻域特征 → 向量
  - 入度/出度
  - 所属域 (one-hot)
  - 变量类别 (fundamental/geometric/quantum/derived)
  - 可达熵的步数
  - 下游分支数
  - 上游根数

因果链 → 聚合变量的向量 → 链向量
  - 序列中变量的平均向量
  - max/avg 深度
  - 域转移序列编码

类似度 → cosine(链向量_A, 链向量_B)
  - 替代手工的 _chain_similarity()
```

## 第二阶段: 学习优化

在因果图上做 self-supervised 学习:

```
正样本: 已知的跨域类比 (KK, 耗散≈退相干)
负样本: 随机跨域配对
目标: 学习嵌入使得已知类比在向量空间中靠近
```

## 第三阶段: 主动发现

用学习到的嵌入替代当前的 `_chain_similarity()`:

```
当前: 手工规则 (长度±1给0.2, 终点类型匹配给0.3...)
目标: 向量余弦相似度 (从因果图结构中学到的)
```
